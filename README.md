# Mine detector using sonar data
Want a submarine ride? Use my code to save your ass from mines. You need a working sonar though.
(A school project)

## Uses libraries:
 * Scikit Learn
 * NumPy

## Algorithms:
 * Principle component analysis
 * Downsampling
 * K-nearest neighbor
 * Logistic regression
 * Gaussian classification
 * Multi layer perceptron
